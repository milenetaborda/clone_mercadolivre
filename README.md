## Como rodar a aplicação

No terminal, clone o projeto:
```
git clone https://github.com/milenetaborda/clone_mercadolivre.git
```

Entre na pasta do projeto:
```
cd clone_mercadolivre
```

Instale as dependências:
```
npm install
```

Execute a aplicação:
```
npm start 
```
